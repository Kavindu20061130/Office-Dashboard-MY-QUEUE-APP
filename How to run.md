# How to Run the Project

You can run this project in 3 different ways. Choose the easiest one for you.

---

# Method 1 - Using Terminal (Manual Setup)

## Steps

Step 1 - Go to project folder (after extracting ZIP)
cd path\to\your\project

Step 2 - Create virtual environment
python -m venv .venv

Step 3 - Activate virtual environment
.venv\Scripts\activate

Step 4 - Install required packages
pip install -r requirements.txt

Step 5 - Run the application
python app.py

## Open in Browser
http://127.0.0.1:5000

---

# Method 2 - Using VS Code (Terminal Inside VS Code)

## Steps

Step 1 - Download and extract the project

Step 2 - Open Visual Studio Code  
Click File → Open Folder → Select project folder

Step 3 - Open terminal  
Click Terminal → New Terminal

Step 4 - Create virtual environment
python -m venv .venv

Step 5 - Activate virtual environment
.venv\Scripts\activate

Step 6 - Install dependencies
pip install -r requirements.txt

Step 7 - Run the application
python app.py

## Open in Browser
http://127.0.0.1:5000

---

# Method 3 - VS Code Easy Method (Top Bar)

## Steps

Step 1 - Open project in VS Code  
Click File → Open Folder → Select project folder

Step 2 - Create virtual environment  
Click "Select Interpreter" (top right)  
Click "Create Virtual Environment"  
Choose "Venv"  
Select Python version  

Step 3 - Open terminal  
Click Terminal → New Terminal  
(Venv will activate automatically)

Step 4 - Install dependencies
pip install -r requirements.txt

Step 5 - Run the application
python app.py

## Open in Browser
http://127.0.0.1:5000
